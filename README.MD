## 介绍
### 简介
- 本仓库是基于[vuepress-theme-vdoing](https://github.com/xugaoyi/vuepress-theme-vdoing) 搭建的知识库兼博客的个人静态网站。
- 搭建的目的主要是有三：
	- 1）以输出倒逼输入，希望能够通过个人平台撰写一些有价值的内容；
	- 2）将自己的知识体系化、结构化，方便自己的回顾和深入理解所思所想的内容；
	- 3）记录自己观点想法，希望结交到更多有趣的朋友。
